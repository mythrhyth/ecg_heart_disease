
# ECG Arrhythmia Classification (Deep Learning)

This project implements an attention-based BiLSTM model for heartbeat classification using the MIT-BIH Arrhythmia Dataset.

## 🔹 Highlights

- Multi-channel ECG input (Raw + Smoothed + Derivative)

- Bidirectional LSTM for temporal modeling

- Attention mechanism for interpretability

- Focal loss & class weights for imbalance handling

- Explainability via saliency maps and Integrated Gradients


## Application

Designed for automated, interpretable ECG analysis in clinical monitoring systems.
